# 🎓 StudyFlow Web

StudyFlow Web은 학생이 오늘 할 일, 집중 타이머, 공부 기록, 일정, 시간표를 한곳에서 관리할 수 있는 공부 관리 웹앱입니다.<br>
설치 없이 브라우저에서 바로 사용할 수 있고, 모든 데이터는 내 브라우저의 `localStorage`에 저장됩니다.

🌐 배포 링크: [https://studyflow.mjiho.com](https://studyflow.mjiho.com)

## ✨ 프로젝트 소개

StudyFlow Web은 "오늘 공부를 바로 시작하고, 끝나면 기록까지 남기는 앱"을 목표로 만든 정적 웹앱입니다.

- 서버 없음
- 회원가입 없음
- 외부 라이브러리 없음
- `index.html` 더블클릭 실행 가능
- 모바일/PC 모두 하단 네비게이션으로 앱처럼 이동

PC에서는 왼쪽에 오늘 브리핑 대시보드가 고정되고, 오른쪽에는 선택한 기능 섹션만 표시됩니다. 모바일에서는 화면을 많이 스크롤하지 않도록 하단 네비바 중심의 앱 형태로 구성했습니다.

## 🚀 실행 방법

### 배포 사이트에서 사용하기

1. [https://studyflow.mjiho.com](https://studyflow.mjiho.com)에 접속합니다.
2. 하단 네비바에서 `오늘`, `타이머`, `기록`, `일정`, `설정`을 선택합니다.
3. 공부 목록, 타이머, 일정, 시간표를 바로 등록해 사용합니다.

### 로컬에서 사용하기

1. 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 더블클릭합니다.
3. 브라우저가 열리면 바로 사용할 수 있습니다.

## 🧭 화면 구성

- 🏠 `오늘`: 오늘 요약, 공부 목록, 목표 시간, 오늘 메모
- ⏱️ `타이머`: 뽀모도로 타이머, 집중 모드, 타이머 설정, 과목 관리
- 📊 `기록`: 주간 리포트, 과목별 공부 시간, 수동 공부 기록
- 📅 `일정`: 캘린더 일정, D-day, 주간 시간표
- ⚙️ `설정`: 데이터 백업/복원, Google Calendar 연동, 테마 안내

## 💡 주요 기능

### ✅ 오늘의 공부 목록

- 과목과 할 일을 추가할 수 있습니다.
- 할 일 왼쪽 체크박스를 누르면 작은 폭죽 효과가 나오고, 효과가 끝난 뒤 완료된 할 일이 자동으로 정리됩니다.
- 과목별 색상이 목록에 표시되어 어떤 공부인지 빠르게 구분할 수 있습니다.
- 오늘 남은 할 일 개수는 오늘 섹션과 PC 왼쪽 브리핑에 함께 표시됩니다.

### 📝 오늘의 메모

- 오늘 섹션의 메모와 PC 왼쪽 대시보드 메모가 실시간으로 동기화됩니다.
- 왼쪽 대시보드에서도 바로 메모를 편집할 수 있습니다.
- 입력 즉시 자동 저장되며, 날짜별로 따로 보관됩니다.

### ⏱️ 뽀모도로 타이머

- 기본값은 `25분 공부 / 5분 휴식`입니다.
- 공부 시간은 1-120분, 휴식 시간은 1-60분으로 직접 설정할 수 있습니다.
- 현재 세션 과목을 선택하면 완료 기록이 해당 과목으로 저장됩니다.
- 자동 전환을 켜면 공부와 휴식 라운드가 이어서 시작됩니다.
- 집중 모드에서는 타이머, 현재 과목, 상태, 조작 버튼만 크게 보여줍니다.

### 📊 공부 기록

- 오늘 총 공부 시간, 이번 주 총 공부 시간, 이번 주 평균 시간을 확인할 수 있습니다.
- 타이머가 공부 세션을 끝내면 자동으로 기록됩니다.
- 직접 공부 기록을 추가하거나 수정/삭제할 수도 있습니다.
- 과목별 공부 시간이 따로 집계됩니다.

### 🎯 목표 시간과 Streak

- 오늘 목표 공부 시간을 설정할 수 있습니다.
- 목표 대비 진행률이 퍼센트와 진행률 바로 표시됩니다.
- 공부 기록이 있는 날을 기준으로 연속 공부 일수 Streak를 계산합니다.
- 하루라도 공부 기록이 없으면 Streak는 끊깁니다.

### 📅 일정과 D-day

- 캘린더에 공부, 과제, 시험, 기타 일정을 추가할 수 있습니다.
- 날짜별 일정 개수를 월간 캘린더에서 바로 확인할 수 있습니다.
- 시험 이름과 날짜를 입력하면 D-day가 자동 계산됩니다.
- 가까운 D-day는 PC 왼쪽 브리핑에도 크게 표시됩니다.

### 🗓️ 시간표

- 요일, 시작 시간, 종료 시간, 과목, 메모를 입력해 시간표를 만들 수 있습니다.
- 시간표는 주간 캘린더 형태로 표시됩니다.
- 과목 색상이 시간표 블록에 반영되어 한눈에 보기 쉽습니다.
- 시간표 항목은 수정/삭제할 수 있습니다.

### 🔗 Google Calendar 연동

설정 섹션에서 Google Calendar 연동을 사용할 수 있습니다.

- Google OAuth Client ID를 저장합니다.
- Google 로그인 후 일정 가져오기/내보내기를 사용할 수 있습니다.
- 가져오기는 현재 보고 있는 월의 Google Calendar 일정을 StudyFlow 일정으로 추가합니다.
- 내보내기는 StudyFlow의 일정과 시험 D-day를 Google Calendar에 종일 일정으로 등록합니다.

정적 웹앱이기 때문에 Google 연동을 쓰려면 직접 만든 Google OAuth Client ID가 필요합니다.<br>
Google Cloud Console에서 OAuth 클라이언트를 만들고, 승인된 리디렉션 URI에 `https://studyflow.mjiho.com/`을 등록해 주세요.

### 🌙 다크모드

- 라이트/다크 모드를 전환할 수 있습니다.
- 선택한 테마는 자동 저장되어 다음 접속 때도 유지됩니다.

### 💾 데이터 백업/복원

- 현재 브라우저에 저장된 StudyFlow 데이터를 JSON 파일로 백업할 수 있습니다.
- 백업 파일을 다시 선택해 데이터를 복원할 수 있습니다.

## 💾 localStorage 저장 방식

모든 데이터는 브라우저의 `localStorage`에 `studyflow-web-state`라는 키로 저장됩니다.

주요 저장 데이터:

- `tasks`: 오늘 공부 목록
- `sessions`: 공부 시간 기록
- `subjects`: 과목 목록, 과목 색상, 과목별 목표 시간
- `schedules`: 캘린더 일정
- `timetable`: 시간표 목록
- `googleCalendar`: Google Calendar Client ID, 액세스 토큰, 연결 상태
- `goalMinutes`: 오늘 목표 공부 시간
- `exams`: 시험 D-day 목록
- `memoByDate`: 날짜별 메모
- `theme`: 선택한 테마
- `timerSettings`: 공부/휴식 시간, 자동 전환, 선택 과목
- `lastGoalCelebratedDate`: 목표 달성 알림 표시 날짜

같은 브라우저에서 새로고침하거나 다시 접속해도 데이터가 유지됩니다.<br>
단, 브라우저 저장 데이터를 삭제하면 StudyFlow 데이터도 함께 삭제됩니다.
