# 📚 StudyFlow Web

StudyFlow Web은 학생이 하루 공부 흐름을 관리할 수 있는 올인원 공부 앱입니다.  
오늘 할 일, 뽀모도로 타이머, 공부 기록, 과목 관리, 캘린더 일정, 시험 D-day, 시간표, 메모까지 한 곳에서 사용할 수 있어요.

👉 배포 링크: [https://studyflow.mjiho.com](https://studyflow.mjiho.com)

설치, 로그인, 서버 없이 사용할 수 있고 모든 데이터는 브라우저의 `localStorage`에 저장됩니다.

## ✨ 프로젝트 소개

공부할 때 필요한 기능은 많지만, 화면이 복잡하면 오히려 쓰기 어렵습니다.  
StudyFlow Web은 기능을 `오늘`, `타이머`, `기록`, `일정`, `설정` 섹션으로 나누어 모바일에서도 앱처럼 빠르게 이동할 수 있게 만들었습니다.

## 🚀 실행 방법

### 배포된 사이트에서 사용하기

1. [https://studyflow.mjiho.com](https://studyflow.mjiho.com)에 접속합니다.
2. 하단 또는 상단 네비게이션으로 원하는 기능을 선택합니다.
3. 공부 목록, 타이머, 기록, 일정, 시간표를 바로 사용할 수 있습니다.

### 로컬에서 사용하기

1. 프로젝트 폴더를 내려받습니다.
2. `index.html` 파일을 더블클릭합니다.
3. 브라우저가 열리면 바로 사용할 수 있습니다.

별도 서버, 로그인, 프레임워크, 외부 라이브러리는 필요하지 않습니다.

## 🧭 화면 구성

- `오늘`: 오늘 요약, 공부 목록, 목표 시간, 오늘의 메모
- `타이머`: 뽀모도로 타이머, 집중 모드, 타이머 설정, 과목 관리
- `기록`: 주간 리포트, 과목별 공부 시간, 수동 기록 추가/수정/삭제
- `일정`: 캘린더 일정, 시험 D-day, 시간표 등록/관리
- `설정`: 데이터 백업/복원, 테마 안내

모바일에서는 한 섹션씩 보여주고 하단 네비게이션으로 이동하도록 구성해 스크롤 부담을 줄였습니다.

## 🧩 주요 기능

### ✅ 오늘의 공부 목록

- 과목과 할 일을 입력해 오늘 공부할 내용을 추가할 수 있습니다.
- 완료한 항목은 체크할 수 있고 완료 항목은 아래쪽으로 정리됩니다.
- 완료/전체 개수를 카드 상단에서 확인할 수 있습니다.
- 필요 없는 항목은 삭제할 수 있습니다.
- 과목별 색상이 함께 표시되어 리스트를 빠르게 구분할 수 있습니다.

### 🏁 오늘 요약

- 오늘 총 공부 시간
- 목표 진행률
- 남은 할 일 개수
- 가장 가까운 D-day

앱을 열자마자 오늘 상태를 빠르게 확인할 수 있습니다.

### ⏱️ 뽀모도로 타이머

- 기본값은 `25분 공부 / 5분 휴식`입니다.
- 공부 시간은 1-120분, 휴식 시간은 1-60분 사이에서 직접 설정할 수 있습니다.
- 시작, 일시정지, 초기화를 지원합니다.
- 공부 중인지 휴식 중인지 상태가 표시됩니다.
- 타이머가 끝나면 공부 세션이 자동 저장됩니다.
- 다음 공부/휴식 라운드를 자동으로 시작할지 선택할 수 있습니다.

### 🎯 집중 모드

- 타이머만 크게 볼 수 있는 집중 모드 화면을 제공합니다.
- 현재 과목, 공부/휴식 상태, 남은 시간을 크게 확인할 수 있습니다.
- 공부할 때 다른 정보가 시야에 들어오지 않도록 도와줍니다.

### 📚 과목 관리

- 자주 공부하는 과목을 미리 등록할 수 있습니다.
- 과목별 색상과 일일 목표 시간을 설정할 수 있습니다.
- 등록한 과목은 할 일, 타이머, 수동 기록, 시간표에서 선택할 수 있습니다.
- 아직 기록에 사용되지 않은 과목은 삭제할 수 있습니다.

### 📊 주간 리포트

- 오늘 총 공부 시간, 이번 주 총 공부 시간, 이번 주 평균 공부 시간을 보여줍니다.
- 월~일 기준의 간단한 막대 그래프로 날짜별 공부량을 확인할 수 있습니다.
- 오늘 공부한 시간을 과목별로 나누어 보여줍니다.
- 연속 공부 Streak를 표시합니다.

### ✍️ 공부 기록 직접 추가/수정

- 타이머를 켜지 못한 공부 시간도 수동으로 추가할 수 있습니다.
- 날짜, 과목, 공부 시간(분)을 입력해 기록합니다.
- 잘못 저장된 기록은 수정하거나 삭제할 수 있습니다.
- 타이머 완료 기록과 수동 기록은 같은 방식으로 합산됩니다.

### 🗓️ 캘린더 일정

- 날짜별로 공부, 과제, 시험, 기타 일정을 추가할 수 있습니다.
- 월별 캘린더에서 일정 개수를 확인할 수 있습니다.
- 선택한 날짜의 일정을 수정하거나 삭제할 수 있습니다.
- 시험 D-day도 캘린더 일정과 함께 표시됩니다.

### 🔥 시험 D-day

- 시험 이름과 날짜를 입력해 D-day를 관리할 수 있습니다.
- 남은 날짜가 `D-10`, 시험 당일은 `D-day`, 지난 시험은 `D+1`처럼 표시됩니다.
- 시험이 7일 이내로 가까워지면 강조 표시됩니다.
- 시험 일정은 수정하거나 삭제할 수 있습니다.

### 🕘 시간표 등록/관리

- 요일, 시작 시간, 종료 시간, 과목, 메모를 입력해 시간표를 등록할 수 있습니다.
- 요일별로 시간표가 정리되어 표시됩니다.
- 등록한 시간표는 수정하거나 삭제할 수 있습니다.
- 과목 색상이 시간표 블록에 함께 반영됩니다.

### 📝 오늘의 메모

- 오늘 공부하면서 기억할 내용이나 해야 할 일을 자유롭게 적을 수 있습니다.
- 입력하는 즉시 자동 저장됩니다.
- 날짜별로 메모가 저장됩니다.

### 🌙 다크모드

- 라이트 모드와 다크 모드를 전환할 수 있습니다.
- 선택한 테마는 저장되어 다음에 열어도 유지됩니다.

### 💬 저장 피드백

- 주요 저장/삭제 동작 후 작은 토스트 메시지로 결과를 알려줍니다.
- 목표 공부 시간을 달성하면 축하 메시지가 표시됩니다.

### 💾 데이터 백업/복원

- 현재 브라우저에 저장된 StudyFlow 데이터를 JSON 파일로 백업할 수 있습니다.
- 백업 파일을 다시 선택해 데이터를 복원할 수 있습니다.
- 브라우저 저장 데이터를 지우기 전에 백업해두면 안전합니다.

## 🛠️ 사용 기술

외부 라이브러리 없이 순수 HTML, CSS, JavaScript만 사용했습니다.

## 💾 localStorage 저장 방식

모든 데이터는 브라우저의 `localStorage`에 `studyflow-web-state`라는 키로 저장됩니다.

저장되는 주요 데이터는 다음과 같습니다.

- `tasks`: 오늘의 공부 목록
- `sessions`: 완료된 공부 세션 기록
- `subjects`: 과목 목록과 과목별 색상/목표 시간
- `schedules`: 캘린더 일정
- `timetable`: 시간표 목록
- `goalMinutes`: 오늘 목표 공부 시간
- `exams`: 시험 D-day 목록
- `memoByDate`: 날짜별 메모
- `theme`: 선택한 테마
- `timerSettings`: 타이머 설정
- `lastGoalCelebratedDate`: 목표 달성 축하 표시 날짜

같은 브라우저에서 새로고침하거나 브라우저를 껐다 켜도 데이터가 유지됩니다.  
단, 브라우저 저장 데이터를 삭제하면 StudyFlow Web 데이터도 함께 삭제됩니다.
