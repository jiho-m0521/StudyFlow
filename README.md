# 📚 StudyFlow Web

StudyFlow Web은 학생이 하루 공부 흐름을 가볍게 관리할 수 있는 웹앱입니다.  
오늘 할 일, 뽀모도로 타이머, 공부 시간 기록, 목표 시간, 시험 D-day, 메모를 한 화면에서 확인할 수 있어요.

👉 배포 링크: [https://studyflow.mjiho.com](https://studyflow.mjiho.com)

설치, 로그인, 서버 없이 사용할 수 있고 모든 데이터는 브라우저의 `localStorage`에 저장됩니다.

## ✨ 프로젝트 소개

공부할 때 가장 중요한 건 “지금 무엇을 해야 하는지”와 “얼마나 집중했는지”를 바로 아는 것입니다.  
StudyFlow Web은 복잡한 기능보다 매일 쓰기 쉬운 공부 대시보드를 목표로 만들었습니다.

v2에서는 타이머와 오늘 할 일을 더 크게 보여주는 집중 대시보드 디자인으로 바뀌었고, 공부/휴식 시간 설정, 자동 다음 라운드 시작, 세션 과목 직접 선택 기능이 추가되었습니다.

## 🚀 실행 방법

### 배포된 사이트에서 사용하기

1. [https://studyflow.mjiho.com](https://studyflow.mjiho.com)에 접속합니다.
2. 바로 공부 목록과 타이머를 사용할 수 있습니다.

### 로컬에서 사용하기

1. 프로젝트 폴더를 내려받습니다.
2. `index.html` 파일을 더블클릭합니다.
3. 브라우저가 열리면 바로 사용할 수 있습니다.

별도 서버, 로그인, 프레임워크, 외부 라이브러리는 필요하지 않습니다.

## 🧩 주요 기능

### ✅ 오늘의 공부 목록

- 과목과 할 일을 입력해 오늘 공부할 내용을 추가할 수 있습니다.
- 과목별 색상을 선택해 리스트를 더 쉽게 구분할 수 있습니다.
- 완료한 항목은 체크하고, 필요 없는 항목은 삭제할 수 있습니다.
- 오늘 날짜의 할 일만 보여주기 때문에 하루 계획에 집중하기 좋습니다.

### ⏱️ 뽀모도로 타이머

- 기본값은 `25분 공부 / 5분 휴식`입니다.
- 공부 시간은 1~120분, 휴식 시간은 1~60분 사이에서 직접 설정할 수 있습니다.
- 시작, 일시정지, 초기화를 지원합니다.
- 공부 중인지 휴식 중인지 상태가 화면에 표시됩니다.
- 타이머가 끝나면 안내 문구가 표시되고, 공부 세션이 기록됩니다.

### 🎯 집중 모드

- 타이머만 크게 볼 수 있는 집중 모드 화면을 제공합니다.
- 현재 과목, 공부/휴식 상태, 남은 시간을 크게 확인할 수 있습니다.
- 공부 중 다른 카드가 시야에 들어오지 않게 하고 싶을 때 유용합니다.

### 📌 세션 과목 선택

- 오늘 공부 목록에 있는 과목이 타이머의 현재 과목 선택창에 자동으로 나타납니다.
- 타이머가 끝나면 선택한 과목으로 공부 시간이 저장됩니다.
- 과목을 선택하지 않으면 `기타`로 기록됩니다.

### 🔁 자동 다음 라운드

- 다음 공부/휴식 라운드를 자동으로 시작할지 선택할 수 있습니다.
- 기본값은 OFF입니다.
- 자동 시작을 켜면 공부가 끝난 뒤 휴식 타이머가, 휴식이 끝난 뒤 공부 타이머가 바로 시작됩니다.

### 📊 공부 시간 기록

- 오늘 총 공부 시간을 확인할 수 있습니다.
- 이번 주 총 공부 시간도 함께 볼 수 있습니다.
- 오늘 공부한 시간을 과목별로 나누어 보여줍니다.
- 공부 세션이 끝날 때마다 기록이 자동 저장됩니다.

### 🏁 오늘 목표 시간

- 오늘 목표 공부 시간을 설정할 수 있습니다.
- 목표 대비 진행률이 퍼센트와 진행률 바로 표시됩니다.
- 공부 시간이 쌓이면 진행률도 자동으로 갱신됩니다.

### 🔥 연속 공부 Streak

- 며칠 연속 공부했는지 표시합니다.
- 오늘부터 하루씩 거꾸로 확인해서 공부 기록이 없는 날이 나오면 streak가 끊깁니다.
- 꾸준히 공부하는 습관을 확인하는 데 도움이 됩니다.

### 🗓️ D-day

- 시험 이름과 날짜를 입력해 D-day를 관리할 수 있습니다.
- 남은 날짜가 `D-10`, 시험 당일은 `D-day`, 지난 시험은 `D+1`처럼 표시됩니다.
- 시험 일정은 수정하거나 삭제할 수 있습니다.

### 📝 오늘의 메모

- 오늘 공부하면서 기억할 내용이나 해야 할 일을 자유롭게 적을 수 있습니다.
- 입력하는 즉시 자동 저장됩니다.
- 날짜별로 메모가 저장됩니다.

### 🌙 다크모드

- 라이트 모드와 다크 모드를 전환할 수 있습니다.
- 선택한 테마는 저장되어 다음에 열어도 유지됩니다.

### 💬 랜덤 동기부여 문구

- 앱을 열 때마다 공부 동기부여 문구가 랜덤으로 표시됩니다.
- `새 문구 보기` 버튼으로 다른 문구를 바로 볼 수 있습니다.

## 💾 localStorage 저장 방식

모든 데이터는 브라우저의 `localStorage`에 `studyflow-web-state`라는 키로 저장됩니다.

저장되는 주요 데이터는 다음과 같습니다.

- `tasks`: 오늘의 공부 목록
- `sessions`: 완료된 공부 세션 기록
- `goalMinutes`: 오늘 목표 공부 시간
- `exams`: 시험 D-day 목록
- `memoByDate`: 날짜별 메모
- `theme`: 선택한 테마
- `timerSettings`: 타이머 설정
  - `studyMinutes`: 공부 시간
  - `breakMinutes`: 휴식 시간
  - `autoStartNext`: 다음 라운드 자동 시작 여부
  - `selectedSubject`: 공부 세션 기록에 사용할 현재 과목

같은 브라우저에서 새로고침하거나 브라우저를 껐다 켜도 데이터가 유지됩니다.  
단, 브라우저 저장 데이터를 삭제하면 StudyFlow Web 데이터도 함께 삭제됩니다.

타이머 설정을 바꿀 때 타이머가 실행 중이면 현재 진행 중인 시간은 바로 바뀌지 않고, 다음 초기화 또는 다음 라운드부터 새 설정이 적용됩니다.
