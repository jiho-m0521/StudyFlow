# StudyFlow Web

StudyFlow Web은 설치 없이 바로 실행할 수 있는 학생용 공부 관리 웹앱입니다. 오늘의 공부 목록, 뽀모도로 타이머, 공부 시간 기록, 목표 시간, 연속 공부 Streak, D-day, 메모, 다크모드를 한 화면에서 관리할 수 있습니다.

v2에서는 타이머와 오늘 할 일을 더 크게 보여주는 집중 대시보드 디자인으로 바뀌었고, 공부/휴식 시간 설정, 자동 다음 라운드 시작, 세션 과목 직접 선택 기능이 추가되었습니다.

## 실행 방법

1. 프로젝트 폴더에서 `index.html` 파일을 더블클릭합니다.
2. 브라우저가 열리면 바로 사용할 수 있습니다.
3. 서버, 로그인, 프레임워크, 외부 라이브러리가 필요하지 않습니다.

## 주요 기능

- 오늘의 공부 목록 추가, 완료 체크, 삭제
- 과목별 색상 선택
- 기본 25분 공부 / 5분 휴식 뽀모도로 타이머
- 공부 시간과 휴식 시간 직접 설정
- 다음 공부/휴식 라운드 자동 시작 설정
- 공부 세션 기록용 현재 과목 직접 선택
- 시작, 일시정지, 초기화
- 집중 모드 화면
- 오늘 총 공부 시간과 이번 주 총 공부 시간 표시
- 과목별 공부 시간 표시
- 오늘 목표 공부 시간과 진행률 바
- 연속 공부 Streak 표시
- 시험 D-day 추가, 수정, 삭제
- 오늘의 메모 자동 저장
- 라이트 / 다크 모드 전환
- 앱 실행 시 랜덤 동기부여 문구 표시

## 사용 기술

- HTML
- CSS
- JavaScript
- localStorage

## localStorage 저장 방식

모든 데이터는 브라우저의 `localStorage`에 `studyflow-web-state`라는 키로 저장됩니다.

저장되는 주요 데이터는 다음과 같습니다.

- `tasks`: 오늘의 공부 목록
- `sessions`: 완료된 공부 세션 기록
- `goalMinutes`: 오늘 목표 공부 시간
- `exams`: 시험 D-day 목록
- `memoByDate`: 날짜별 메모
- `theme`: 선택한 테마
- `timerSettings`: 타이머 설정
  - `studyMinutes`: 공부 시간
  - `breakMinutes`: 휴식 시간
  - `autoStartNext`: 다음 라운드 자동 시작 여부
  - `selectedSubject`: 공부 세션 기록에 사용할 현재 과목

같은 브라우저에서 새로고침하거나 브라우저를 껐다 켜도 데이터가 유지됩니다. 단, 브라우저 저장 데이터를 삭제하면 StudyFlow Web 데이터도 함께 삭제됩니다.

타이머 설정을 바꿀 때 타이머가 실행 중이면 현재 진행 중인 시간은 바로 바뀌지 않고, 다음 초기화 또는 다음 라운드부터 새 설정이 적용됩니다.
